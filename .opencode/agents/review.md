# Review Agent

A specialized agent for code review tasks.

## Description

This agent focuses on reviewing code for:
- Code quality and best practices
- Security vulnerabilities
- Performance issues
- Documentation completeness

## Usage

Use with commands like:
```
*review /ghar-security-check
*review /ghar-stale-check
```

## Capabilities

- Security-focused code analysis
- Style and formatting review
- Architecture and design review
- Documentation review
